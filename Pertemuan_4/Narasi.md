## Naming Convention

Naming convention menurut pemahaman saya merupakan sebuah aturan yang digunakan dalam pemberian nama pada variabel, class, method, dll agar pembacaan nama tersebut mudah dipahami. Pemberian nama yang baik dan benar akan menjadikan program lebih rapi, mudah dibaca, jelas dan tidak membingungkan. Dengan adanya aturan ini, programmer dapat menulis kode yang lebih terstruktur dan konsisten.

## Penamaan Variabel dan Class

Penamaan variabel dan class penting dalam program karena dapat membantu dalam memahami program yang kita buat. Nama yang jelas pada variabel dan class dapat membantu kita untuk lebih mudah dalam memahami, mengingat dan mengetahui kegunaan dari variabel dan class tersebut. Selain itu penamaan variabel dan class juga penting untuk mengurangi kesalahan penulisan sehingga tidak mudah tertukar. Hal ini sangat membantu terutama saat program sudah semakin panjang dan kompleks.

## Operator Aritmatika

Operator artimatika pada java digunakan untuk melakukan operasi perhitungan matematika seperti perkalian, penjumlahan, pembagian, pengurangan hingga modulo. Operator aritmatika ini bekerja dengan cara mengolah nilai menggunakan operator tersebut. Misalnya operator “*” digunakan untuk perkalian, “+” penjumlahan, “-“ pengurangan, “/” pembagian, hingga “%” digunakan untuk modulo (sisa bagi). Dengan adanya operator aritmatika, perhitungan dalam program dapat dilakukan dengan lebih cepat dan praktis.

## Pemahaman dan Penerapan

Pemahaman saya mengenai penerapan pada latihan yang sudah saya kerjakan yaitu saya memahami bahwa penamaan variabel yang baik akan sangat membantu dalam memahami program yang dibuat dan menghindari kesalahan. Selain itu operator artimatika juga sangat membantu dalam mempercepat proses perhitungan matematika. Penggunaan variabel yang benar dalam program juga sangat membantu dalam membedakan setiap data sesuai dengan kebutuhannya. Misalnya, untuk bilangan desimal sebaiknya menggunakan tipe data double atau float, sedangkan untuk bilangan bulat menggunakan int. Dengan memilih tipe variabel yang tepat, program dapat berjalan lebih efisien, hasil perhitungan menjadi lebih akurat, dan risiko kesalahan dapat diminimalkan. Kemudian penggunaan konstanta juga sangat penting untuk menyimpan nilai tetap yang tidak boleh berubah selama program berjalan, seperti nilai “pajak” yang dikerjakan pada soal latihan atau “pi” pada program praktikum.
